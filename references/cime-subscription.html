구독 이벤트

구독 이벤트를 구독하면 채널에서 발생하는 구독을 실시간으로 수신합니다.

구독 엔드포인트: POST /api/openapi/open/v1/sessions/events/subscribe/subscription

필요 Scope: READ:SUBSCRIPTION

이벤트 데이터

{
  "event": "SUBSCRIPTION",
  "data": {
    "channelId": "12345",
    "subscriberChannelId": "67890",
    "subscriberChannelName": "구독자명",
    "month": 3,
    "tierNo": 1,
    "subscriptionMessage": "3개월 구독합니다!",
    "emojis": {}
  }
}
필드 타입 설명
channelId string 구독이 발생한 채널의 ID
subscriberChannelId string 구독자의 채널 ID
subscriberChannelName string 구독자의 채널 이름
month number 연속 구독 개월 수
tierNo number 구독 티어 번호
subscriptionMessage string 구독 메시지
emojis object 이모티콘 토큰과 이미지 URL 매핑
